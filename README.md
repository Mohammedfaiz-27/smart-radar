# SMART RADAR MVP

A real-time social media intelligence platform designed as a Digital Command Center to monitor, analyze, and respond to online narratives.

## Technology Stack

- **Frontend**: Vue.js 3.4+
- **Backend**: Python 3.11+ with UV package manager
- **Database**: MongoDB 7.0+

## Quick Start

### Backend Setup
```bash
cd backend
uv venv
uv pip install -r requirements.txt
uv run python main.py
```

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

## Project Structure

- `/docs/` - All documentation
- `/tests/` - Test and debug scripts
- `/frontend/` - Vue.js application
- `/backend/` - Python backend application

## Features

- Cluster & Keyword Management
- Real-time Data Collection
- Narrative Bank for Response Templates
- Unified Dashboard with Own/Competitor Feeds
- AI-Powered Response Generation
- Real-time Alerting System
